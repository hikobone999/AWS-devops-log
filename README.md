### 🧠 今日の準備ログ（2025/05/11）

#### 🖥️ Linux（Git Bash）練習コマンドログ
- `pwd`：現在のディレクトリを表示
- `mkdir test_folder`：フォルダ作成
- `cd test_folder`：ディレクトリ移動
- `touch test.txt`：空ファイル作成
- `ls`：中身表示
- `rm test.txt`：ファイル削除
- `cd ..`：1つ上へ戻る
- `rmdir test_folder`：空フォルダ削除

📌 手で操作して体に馴染ませるのが目標。数回繰り返して「思考なしで手が動く」状態へ。
